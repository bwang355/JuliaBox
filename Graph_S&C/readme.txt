This folder provides supporting codes for the paper "Solution of Large-Scale Supply Chain Models using Graph Sampling & Coarsening".

* The folder "time_test" contains the code and data files used to derive three levels of upper&lower bounds for large-scale case study. 

* The folder "memory_test" contains the code and data files for testing the memory usage of Gurobi and gSC.

* We recommend use Julia 1.6.3 and Gurobi 9.1 to run all code files.
